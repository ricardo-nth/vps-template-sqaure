## Quick Start

1. Click **Use this template** on GitHub.
2. Clone your new repository.
3. Install dependencies:

   ```bash
   pnpm install
   ```

4. Start the dev server:

   ```bash
   pnpm dev
   ```

5. Open the local URL shown in the terminal and begin editing content.
